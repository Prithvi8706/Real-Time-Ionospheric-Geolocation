This console drives the single-station location (SSL) pipeline. An HF signal launched beyond the horizon reaches the receiver by refracting off the ionosphere, a charged shell of the upper atmosphere a few hundred kilometres up.

From the measured arrival bearing, elevation angle and a condition-aware ionospheric height model, the pipeline back-projects the hop geometry in two passes: receiver, rough transmitter, path midpoint, final transmitter.

Set the intercept parameters and compute. The hop cross-section, map solution and uncertainty bounds will render here from the live response.
